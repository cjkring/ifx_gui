How to get started
------------------

1) Install anaconda 

https://docs.anaconda.com/anaconda/install/mac-os/

2) Open a terminal and cd to this directory

3) Build and activate a conda environent

conda create --name ifx_gui --file conda_env.txt
conda activate ifx_gui

4) plug the Radar module into a USB port

5) run the python app

python3 iqplot.py
